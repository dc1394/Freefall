★これは何？
　空気抵抗のある自由落下系における運動方程式を数値的に解き、結果をGUI画面に表示
　するプログラムです。計算結果をCSVファイルに出力することもできます。

★インストール
　Freefall_x64_(バージョン番号).zip（x64版）またはFreefall_x86_(バージョン番号)
　.zip（x86　版）を任意のフォルダに解凍してください。
　ただし、このソフトを動作させるためには、「Microsoft .NET Framework 4.7」
　（ https://www.microsoft.com/en-us/download/details.aspx?id=55170 からダウンロ
　ード可能）と、「Visual Studio 2017 の Microsoft Visual C++ 再頒布可能パッケー
　ジ」（ https://visualstudio.microsoft.com/ja/downloads/?q=#other-ja からダウ
　ンロード可能）が必要です。なお、Freefall x86版の動作には、「vcredist_x86.exe」
　が、Freefall x64版の動作には「vcredist_x64.exe」が必要となります（OS自体のx86、
　x64バージョンとは関係が無いので注意して下さい。なお当然ですが、Freefall x64版
　は、x86バージョンのWindowsでは動きません。）

★アンインストール
　解凍したフォルダを削除して下さい。
　このソフトはレジストリを変更しません。

★更新履歴

★謝辞
　このソフトウェアのBindaleBaseクラスは、「id:minami_SC」様の「WPFでも
　BindableBaseを使ってINotifyPropertyChangedを実装する
　( http://sourcechord.hatenablog.com/entry/20130303/1362315081 )」というブログ
　記事を、ViewModelBaseクラスは、「id:minami_SC」様の「WPFでの入力値検証・その2 
　～INotifyDataErrorInfoを使ってみる～
　( http://sourcechord.hatenablog.com/entry/2014/06/08/123738 )」というブログ記
　事と、「hugo-sb」様の「WPF データバインディング項目用のヘルパークラス
　( http://qiita.com/hugo-sb/items/dd7da4616400ba57d48c )」というQiitaの記事を、
　最後にEnumBooleanConverterクラスは、「ba」様の「Enum と RadioButton をバインド
　する( http://frog.raindrop.jp/knowledge/archives/002200.html )」というブログ記
　事をそれぞれ参考にさせて頂きました。この場をお借りしまして御礼申し上げます。
